#### \35. Search Insert Position

```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        if (nums == null) return 0;
        int lower = 0, upper = nums.length - 1;
        while (lower < upper) {
            int mid = lower + (upper - lower) / 2;
            if (nums[mid] > target) {
                upper = mid;
            } else if (nums[mid] < target) {
                lower = mid + 1;
            } else {return mid;}
        }
        if (nums[lower] >= target) {
            return lower;
        } else {
            return lower + 1;
        }
    }
}
```
